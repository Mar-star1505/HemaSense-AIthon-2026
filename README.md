## 📋 About

HemaSense is an intelligent consumer profiling system that:
- 📱 Scans QR codes to launch personalised product recommendations
- 🤖 Uses Google Gemini AI to generate custom care routines
- 📊 Tracks consumer preferences in real-time
- 📈 Provides brand managers with analytics dashboard

## 🛠️ Tech Stack

- **Frontend:** Streamlit
- **AI Engine:** Google Gemini 2.5 Flash
- **Data Analytics:** Pandas, Plotly
- **Deployment:** Streamlit Cloud

## 🔧 Local Setup

```bash
# Clone the repository
git clone https://github.com/Mar-star1505/Hemasense-Aithon-2026.git

# Navigate to project
cd Hemasense-Aithon-2026

# Install dependencies
pip install -r requirements.txt

# Create .streamlit/secrets.toml with:
GEMINI_API_KEY = "your-api-key-here"

# Run consumer app
streamlit run app.py

# Run dashboard (new terminal)
streamlit run dashboard.py
